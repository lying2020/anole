## 🚀 快速开始

### 1. 文本生成图像

```bash
cd /home/liying/Desktop/ECCV_2026/anole
python text2image.py -i "a beautiful cat" -b 1 -s ./outputs/test/
```

**参数说明**：
- `-i`: 图像生成提示词
- `-b`: 生成数量（建议先用 1 测试）
- `-s`: 保存目录

### 2. 交错图像-文本生成

```bash
python interleaved_generation.py -i "Introduce Paris with pictures" -s ./outputs/interleaved/
```

**功能**：生成包含文本和图像的混合内容

### 3. 多模态输入推理

首先准备输入文件 `test_input.json`：
```json
[
    {
        "type": "text",
        "content": "What is in this image?"
    },
    {
        "type": "image",
        "content": "assets/stop_sign.png"
    }
]
```

然后运行：
```bash
python inference.py -i inputs/input.json -s ./outputs/inference/
```

## 📊 项目架构总结

### 核心组件

1. **ChameleonInferenceModel** (`chameleon/inference/chameleon.py`)
   - 主推理类
   - 管理模型加载、tokenization、生成

2. **路径配置系统**
   - `constants.py`: 构建完整路径，支持环境变量覆盖

3. **三个主要脚本**
   - `text2image.py`: 纯文本→图像
   - `interleaved_generation.py`: 文本→交错内容
   - `inference.py`: 多模态输入→多模态输出

## ⚠️ 注意事项

1. **内存要求**：7B 模型需要较大 GPU 内存，建议至少 16GB
2. **首次运行**：模型加载需要时间，请耐心等待
